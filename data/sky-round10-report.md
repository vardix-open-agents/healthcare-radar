## Key Findings

### 🚨 CRITICAL INSIGHT: ZERO EUROPEAN PLAYERS

Nova was RIGHT. There are **ZERO European companies** focused on:
- Insurance eligibility verification APIs
- Prior authorization automation
- Real-time claims status checking

This is a **MASSIVE BLUE OCEAN**.

### US Market Leaders (Templates for EU Replication)

**Eligibility Verification:**
1. **Availity** - Nation's largest real-time healthcare network, 2M+ providers
2. **Experian Health** - AI-powered Patient Access Curator
3. **Change Healthcare (Optum)** - Payer-provider transaction infrastructure
4. **Waystar** - AI-powered payments platform, 60% of US patient population
5. **Eligible** - API-first eligibility verification platform

**Prior Authorization:**
1. **CoverMyMeds** - Largest prior auth platform (McKesson)
2. **Surescripts** - Intelligent Prior Authorization, 30.5B transactions
3. **eviCore** - Specialty prior auth for radiology/cardiology
4. **AGS Health** - Intelligent Authorization AI platform
5. **CoHelm** - AI prior auth for health plans

**Claims & RCM:**
1. **Cognizant TriZetto** - 4.4B annual payer-provider transactions
2. **R1 RCM** - Revenue Operating System with AI agents
3. **GeBBS** - Technology-enabled RCM services

### European Infrastructure (Foundation Exists, Automation Missing)

| Country | Infrastructure | Missing Layer |
|---------|---------------|---------------|
| **Germany** | Telematics infrastructure, eGK cards | No eligibility APIs, no prior auth automation |
| **Netherlands** | VECOZO network, DBC/DOT billing | No real-time eligibility, no prior auth tools |
| **France** | Carte Vitale, FSE system | No prior auth automation |
| **EU-wide** | National MPIs, GDPR frameworks | No cross-border eligibility verification |
