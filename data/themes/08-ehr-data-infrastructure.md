# Theme: EHR & Data Infrastructure

## Summary
Platforms and APIs that solve the EHR integration problem, enable data portability, and connect fragmented healthcare systems.

## Companies
- **Leyr** (Sweden) - Unified API platform connecting to multiple EHRs through single integration. 92% reduction in development time. 2-week average integration. No data storage, pass-through architecture. ISO 27001, GDPR, HIPAA compliant.
- **Wello** (Netherlands) - Personal health record platform. Patient-controlled data with provider sharing.
- **Care to Translate** (Norway) - Healthcare translation platform. Deployed in Helse Sør-Øst (Norway's largest health region). Medical terminology accuracy.
